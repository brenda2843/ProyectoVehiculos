# Sistema de Registro de Vehículos - SENA

## Descripción
Sistema web desarrollado en Java con JSP y Servlets para el registro y consulta de vehículos.
Proyecto académico del programa Análisis y Desarrollo de Software - SENA.

## Tecnologías utilizadas
- Java
- JSP (JavaServer Pages)
- Servlets
- Apache Tomcat 10
- HTML5 y CSS3
- Git y GitHub

## Funcionalidades
- Inicio de sesión con autenticación (usuario: admin, contraseña: 1234)
- Registro de vehículos con validación de datos
- Validación de placa duplicada y cédula duplicada
- Validación de formato de placa (ABC123 o ABC12F)
- Diseño responsive

## Estructura del proyecto
- auth.jsp — Página de inicio de sesión
- index.jsp — Página principal
- login.jsp — Formulario de registro de vehículos
- resultado.jsp — Muestra los datos registrados
- AuthServlet.java — Servlet de autenticación
- VehiculoServlet.java — Servlet de registro de vehículos
